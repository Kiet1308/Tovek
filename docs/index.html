Six recovery layers. Four guarantees.

Version 0.7 rebuilds the pipeline from SSA destruction to final AST cleanup. Read P0–P5 in order: correctness first, source recovery second, cleanup last. The final four rows are release guarantees.

LAYER P0

Semantic correctness

SSA and phi nodes, loop-carried state, captures, upvalues and multi-value calls are rebuilt without stale snapshots, accidental globals or swallowed evaluations.

phi · upvalues · multivalue
LAYER P1

Optimizer reversal

Statement helpers, expression helpers, CPS continuations and repeated branch tails erased by Luau -O2 return as concise calls when scope and capture proofs allow it.

-O2 copies → helper
LAYER P2

Naming & source recovery

Locals, parameters, services, modules, callbacks and React roles are inferred from usage and consistent call sites, with lexical shadowing protection and intact long names.

v9 → connection
LAYER P3

Structured control flow

Guard continue, elseif, repeat-style loops, ancestor walks and compound assignments are recovered with goto, loop-ownership and NaN safety gates.

branches → structured Luau
LAYER P4

Tables & React/UI

Exploded literals, computed event keys, props, children, callbacks and nested createElement trees are rebuilt leaf-to-root without changing evaluation order.

t.X ×N → { … }

Safe cleanup & idioms

Capture-aware aliases, dead stores, void returns and loop exits are cleaned up; module shapes, callback assignments and exact Roblox constants are restored without deleting throws or side effects.
